## O que aconteceu
Agente `fabric-engineer` entrou em loop de retentativas (>3) sem progresso após sequência de `TodoWrite` e `Write` para arquivos `prd_poc_itausa_fabric_medallion_docx.md` e `spec_poc_itausa_fabric_medallion_docx.md`; última tool truncada (`TodoW`) sugere falha na escrita do todo.

## Causa raiz
Atualização de todo list (`TodoWrite`) ocorrendo sem validação de estado anterior, possivelmente sobrescrevendo ou corrompendo o buffer de tarefas e impedindo o agente de reconhecer progresso.

## Padrão para evitar
Sempre ler o estado atual do `Todo` antes de `TodoWrite`; nunca emitir `TodoWrite` consecutivo sem verificar conclusão da tarefa anterior.
