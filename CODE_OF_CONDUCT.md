# Code of Conduct

This project adopts the **[Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)** by reference.

The full canonical text is maintained at the link above. By participating in this project — as a contributor, maintainer, issue author, reviewer, or user — you agree to abide by it.

## Reporting

Concerns can be reported via either channel:

- **Email**: [thomaz.rossito@terra.com.br](mailto:thomaz.rossito@terra.com.br)
- **GitHub Private Vulnerability Reporting**: [Open a private advisory](https://github.com/ThomazRossito/ai-data-agents/security/advisories/new) (use only for issues that combine conduct with security concerns).

All reports will be reviewed and investigated promptly and fairly. The project commits to:

| Step | Target |
|---|---|
| Acknowledgement of report | within **72 hours** |
| Initial assessment | within **7 days** |
| Resolution / decision | within **30 days** for non-urgent matters |

All reporters are treated with confidentiality and respect. The identity of a reporter is never disclosed without their explicit consent.

## Scope

This Code applies to all project spaces, including:

- GitHub repository (issues, pull requests, discussions, commits, code review).
- Project-related communication channels (email, video calls).
- Public spaces when an individual is representing the project (talks, blog posts, social media tied to the project).

## Enforcement

The maintainer ([@ThomazRossito](https://github.com/ThomazRossito)) is responsible for clarifying and enforcing standards. Possible responses include — and are not limited to — a private conversation, a public correction, a temporary or permanent removal from project spaces, or revocation of contribution privileges.

Enforcement decisions follow the **[Contributor Covenant Enforcement Ladder](https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement-guidelines)**:

1. Correction (private written warning)
2. Warning (consequences for continued behavior)
3. Temporary ban
4. Permanent ban

The specific response is chosen at the maintainer's discretion based on severity, prior history, and impact.

## Disagreements with enforcement

Anyone who believes they have been unfairly subject to an enforcement decision may appeal by emailing the maintainer with the subject line `[CoC Appeal]`. Appeals are reviewed within 14 days.
